

# Project Name: FoodOrderApp

---

## Description
  **FoodOrderApp:** 
 — это мобильное приложение на платформе iOS, представляющее собой локальный агрегатор и сервис доставки из сети ресторанов быстрого питания в городе Минске. Программный комплекс обеспечивает полный цикл обслуживания: от сквозной регистрации и авторизации пользователей с помощью встроенной реляционной СУБД SQLite3 до интерактивного выбора и кастомизации состава ингредиентов блюд. Встроенная интерактивная корзина поддерживает реактивный пересчет стоимости позиций на лету и гибкое управление составом чека без удаления объектов из памяти. Кроме того, геоинформационный модуль на базе MapKit и CoreLocation автоматически вычисляет кратчайшее расстояние до ближайшего ресторана сети на основе текущих GPS-координат пользователя.

## Installation
Для локальной сборки и запуска приложения на симуляторе iOS или физическом устройстве Apple выполните следующие шаги:

1. **Клонирование репозитория:**
   Склонируйте репозиторий с исходным кодом проекта на ваш локальный компьютер под управлением macOS, используя терминал:
   ```bash
   git clone [https://github.com/your-repository/FoodOrderApp.git](https://github.com/your-repository/FoodOrderApp.git)
   ```
  
2. **Открытие проекта в среде разработки:**
  Запустите среду разработки Xcode и откройте файл проекта, выбрав его в меню `File -> Open` или дважды кликнув по файлу конфигурации в корне каталога:
   ```text
   FoodOrderApp/FoodOrderApp.xcodeproj
   ```


3. **Выгрузка зависимостей:**
Проект разработан на базе нативных фреймворков Apple (**SwiftUI**, **Combine**, **CoreLocation**, **MapKit**) и встроенной системной библиотеки **SQLite3**, поэтому установка дополнительных сторонних зависимостей (через CocoaPods или Swift Package Manager) не требуется.
4. **Выбор целевого устройства:**
В верхней панели Xcode (меню Target/Scheme) выберите желаемый симулятор смартфона (например, *iPhone 15* или новее) либо подключите ваше физическое iOS-устройство через кабель.
5. **Инициализация и запуск:**
Нажмите комбинацию клавиш `Cmd + R` или кликните на иконку **Play** в левом верхнем углу панели Xcode. Произойдет автоматическая компиляция исходного кода, инициализация базы данных `foodorder.sqlite` в каталоге документов приложения (`.documentDirectory`) и старт интерфейса на выбранном устройстве.

## Usage

После успешного запуска приложения взаимодействие с системой осуществляется по следующему сценарию:

* **Аутентификация и валидация:**
При первом запуске вас встретит экран `AuthView`. Используйте верхний переключатель (Segmented Control) для перевода формы в режим «Регистрация» (`Signup`). Введите уникальный логин (не менее 3 символов), пароль (не менее 4 символов) и адрес доставки (не менее 5 символов). После успешного создания учетной записи в базе данных SQLite система автоматически перенаправит вас на главную витрину.
* **Работа с меню и модификация блюд:**
На главном экране `MenuView` выберите интересующую вас категорию блюд (Пицца, Бургеры, Закуски или Напитки) с помощью верхней панели категорий. Нажмите на карточку любого блюда — откроется модальное окно настроек `DishDetailView`. Здесь интерфейс автоматически подстроится под тип еды: для пиццы можно выбрать тесто и размер, для бургеров — отметить галочками добавление сыра, лука, томатов или огурцов. Нажмите кнопку «Добавить», чтобы зафиксировать модифицированное блюдо.
* **Управление корзиной (CartView):**
Нажмите на плавающую нижнюю кнопку «Корзина» на главном экране. В открывшемся списке вы можете управлять позициями заказа. Если вы снимите галочку (тумблер) с какого-либо блюда, текст названия автоматически зачеркнется, строка станет полупрозрачной, а общая стоимость заказа мгновенно пересчитается (система исключит эту позицию из суммы заказа без физического удаления объекта из корзины).
* **Выбор геолокации и оформление заказа:**
Внутри корзины нажмите кнопку «Выбрать на карте», чтобы открыть модуль `MapView`. Вы увидите 5 захардкоженных ресторанов сети на карте Минска. Нажмите кнопку **«Выбрать ближайший»** — приложение запросит доступ к GPS, рассчитает по формуле гипотенузы Евклидово расстояние от вашей текущей геопозиции до каждого филиала, плавно переместит карту на самый близкий к вам ресторан и автоматически подставит его адрес в поле доставки.
* **Завершение заказа и фискальный лог:**
Выберите один из 4 способов оплаты (*Онлайн, ЕРИП, Терминал, Наличные*), при необходимости введите текстовый комментарий для курьера и нажмите финальную кнопку «Оформить заказ». Корзина полностью очистится, на экране отобразится анимация успешного действия, а в системную консоль Xcode выведется структурированный фискальный чек с детализацией цен и параметров кастомизации каждого блюда.
